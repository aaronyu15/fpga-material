This page lists some simple but worthwhile projects that others have posted online. Use them to learn how to do cool things, and then adapt those things into your own project work.

## Hackster
[Hackster](https://www.hackster.io/) is a great website since there are lots of people who post not only their projects, but commonly the step-by-step instructions and code that go along with it. Note that projects tend to use different versions of Vivado, so you may have to adapt the instructions towards your particular version if necessary (sometimes this is easy, sometimes it is not). Unless projects are very specific to a board, they can generally be adapted to other boards as well with minor/major modifications. Some good projects to review are listed here:

Useful for learning and prototyping, though for more practical designs it would be better to have the AXIlite interface included in your design directly. |

A very good and useful tutorial if you want to do any sort of data processing alongside the [[Xilinx Terminology#PS|PS]].                                                                                                   |
