    Guides

    The official Design Hubs Landing Page contains an organized set of links that you can use to find relevant user guides and tutorials for you purpose. However, there are a lot of them and it may not immediately be clear where to start or what is useful to you. In this page I organize these documents into tables with brief descriptions to make finding useful content easier.

    Notation: ✅ - Recommended 🤔 - Optional ❌ - Skippable

    The following sections are organized based on relevance to Vivado, Vitis, and specific FPGA development boards.

    Vivado

    Vivado is the main tool that allows you to convert a hardware design (either from hand-coded RTL files or IP blocks) into a bitstream to program the Programmable Logic (PL) side of the FPGA. In Vivado you can perform simulation, synthesis, implementation, and bitstream programming. Aside from simulation, these steps depend on the board that is selected since the board determines the available hardware resources, pins, pinout layout, etc.

    Vivado User Guides and Tutorials

    Vivado user guides give instructions on how to use the various features in Vivado. Typically, each user guide has an accompanying tutorial that goes along with it. It is recommended to run through the tutorial first as it introduces the main content, and then refer to the user guide afterwards to learn the full extent of the topic.

    Vivado tutorials have step-by-step labs that guide you through toy designs to demonstrate parts of the design flow.

    Vitis

    Vitis is a Vivado companion tool that has two main functions: to program embedded designs and to perform High Level Synthesis (HLS). For embedded designs, if the FPGA board contains hard logic, like the Zynq series which have various “CPU”s, then Vitis can be used to program them. This relates to the Processing System (PS) side of the FPGA. For HLS, Vitis can be used to convert C/C++/other code to a hardware implementation. Essentially, it translates high level algorithms it into RTL. This relates to the Programming Logic (PL) side of the FPGA.

    Info

    In addition to the official Design Hubs Landing Page, Vitis has its own landing page UG1416. The main Design Hubs Landing Page has links to the Vitis user guides, but does not seem to link to the tutorials. This is the main difference, aside from a pretty interface.

    PetaLinux

    PetaLinux is a light Linux image based on Yocto that has official support for use on Zynq based boards. In short, you can run Linux on the PS side, and have it interact with your custom programmable logic.

    UsefulUser GuideTitleDescription
    ✅UG1144PetaLinux Reference GuideGuide to installing PetaLinux + some configurations.

    MicroBlaze Processor

    For Xilinx FPGAs (Spartan, Artix, Kintex, Virtex), there is not hard-logic processor like there is on Zynq. Instead, a soft-core processor called the MicroBlaze can be programmed into the FPGA fabric if desired.

    UsefulUser GuideTitleDescription

    ✅UG984MicroBlaze Processor Reference GuideA more detailed guide to using the MicroBlaze processor.
